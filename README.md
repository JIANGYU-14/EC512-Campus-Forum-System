# Campus Forum Website 

* Forum management website based on ASP.NET framework which involving session state and forms based security. 

* Private folder and web.config properly set to deny access to other than authenticated user.

* Users are able to register, post comment, view others comment as well. 

* Session state cookies implemented to keep track of the authecticated user througout the web.

* The web has been tested and optimized using Monkey Test to achieve low latency and high stability.
